# shock-intelligence
Macroeconomic Shock Intelligence Engine

Detecting and explaining macroeconomic shocks using data and AI.

This repository contains a research-oriented system for detecting, ranking,
and interpreting macroeconomic shocks and their spillover effects using
PostgreSQL and Python.

The project focuses on reproducibility, conservative inference, and
transparent analytical design.

Data ingestion scripts, SQL-based analytics, and research briefs are included.
Raw data sources are external and not redistributed.
